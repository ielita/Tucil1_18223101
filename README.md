# Tucil1_18223101

## Program Penyelesaian Queens LinkedIn
Queens adalah permainan yang tersedia pada situs LinkedIn. Permainan ini mengharuskan pemain untuk menempatkan Queens pada papan NxN. Terdapat beberapa aturan utama permainan Queens:
1. Pemain harus menempatkan tepat satu Queen di setiap baris, kolom, dan area warna
2. Dua queen tidak boleh bersentuhan dengan satu sama lain, termasuk secara diagonal


## Requirement dan Instalasi
Bahasa pemrograman yang digunakan adalah Python dengan menggunakan modul standar Python (time dan copy). Program ini dikembangkan pada OS Windows.

## Cara menjalankan
1. Buka terminal, CMD, atau sebagainya
2. Arahkan ke direktori tempat file tersimpan
3. Jalankan perintah python main.py
4. Masukkan nama file board yang terdapat pada folder test ketika diminta pada CLI
5. Program akan menampilkan visualisasi papan, waktu pencarian, dan total iterasi kasus
6. Pemain dapat memilih untuk menyimpan hasil program jika diinginkan
